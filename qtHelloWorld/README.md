# This is a small project to get a button in QT

A minimalistic example of creating a window with a button in QT

## Requisites

```
    sudo apt install qtbase5-dev
```

## Build

```
    cmake -B build
    cmake --build build
```
## Run
```
    ./qtHelloWorld
```
